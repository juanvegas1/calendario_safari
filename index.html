Casos de Éxito Reales

Proyectos similares con impacto medible en empresas comerciales y de servicios.

Sector Industrial / Comercial

Automatización CRM → Dashboard en Tiempo Real

Python API REST SQL Server Power BI .BAT

El CRM corporativo (ForceManager) no permitía conexión directa con Power BI. Desarrollé un sistema que consume la API del CRM automáticamente, alimenta una base de datos SQL mediante tareas programadas (.bat), y Power BI se actualiza con data ya limpia y lista para análisis.

4 horas diarias de trabajo manual eliminadas
Sector Seguros

Consolidación Multi-Aseguradora Automatizada

Python Power Query Excel Power BI

El cierre mensual requería descargar manualmente reportes de múltiples aseguradoras, unificarlos y calcular comisiones de vendedores. Automaticé todo el flujo: Python ingresa a cada plataforma, descarga los Excel, los compila, unifica y limpia. Power BI consume la data final automáticamente.

Cierre mensual reducido de 5 a 2 días
Sector BPO / Servicios

SharePoint → Dashboard Excel Automatizado

SharePoint Power Query Excel VBA

Las bases de datos dispersas en SharePoint dificultaban el análisis consolidado. Implementé conexiones Power Query que unifican y transforman los datos automáticamente, con un dashboard en Excel que se actualiza programáticamente sin intervención manual.

50% reducción tiempo + 30% menos errores
